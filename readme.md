## Dev

```sh
# install dependencies
$ npm install

# support for reloading views, restarting electron
# if app/* and/or index.js, electron/* are changed
$ npm start
```


### Build

```sh
$ npm run build      # all
$ npm run build-osx  # osx(64)
$ npm run build-win  # win(32, 64)
```
